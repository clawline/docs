# Client WeChat / 微信小程序

微信原生小程序客户端，提供与 Web 端对等的聊天体验。

## 它做什么

- 连接 Relay Gateway 或直连 OpenClaw 节点
- 流式消息渲染（逐字显示 AI 回复）
- 思考状态展示（思考中 / 分析中 / 整理中）
- 语音录制（AAC 格式，最长 60 秒）
- 图片发送
- 表情回应、引用回复、消息编辑/删除
- 20+ 斜杠命令
- AI 后续建议
- 多服务器管理
- 深色/浅色模式

## 技术栈

微信小程序原生框架 + glass-easel 高性能引擎

## 关键特性

- **WebSocket 连接池**: 页面切换时复用连接（15 秒宽限期）
- **离线队列**: 最多 50 条待发消息（LRU + wx.Storage）
- **消息持久化**: 每 Agent 最多 200 条本地消息
- **响应式**: 适配多种屏幕尺寸

## 页面结构

| 页面 | 功能 |
|------|------|
| Entry | 路由入口，检查配对状态 |
| Onboarding | 功能展示轮播 |
| Pairing | 服务器连接（URL / QR / 手动输入） |
| Agents | Agent 列表（搜索、多服务器分组） |
| Chat Room | 主聊天界面 |
| Dashboard | 服务器状态监控 |
| Search | 本地全文消息搜索 |
| Profile | 服务器管理、通知设置 |
| Preferences | AI 配置（模型/温度/提示语） |

## Web vs 小程序功能对比

| 功能 | Web | 小程序 |
|------|-----|--------|
| 多服务器管理 | 支持 | 支持 |
| 流式消息 | 支持 | 支持 |
| 表情回应 | 支持 | 支持 |
| 语音输入 | Volc ASR | 原生录音 |
| 斜杠命令 | 24+ | 20+ |
| 离线队列 | 200 条 | 50 条 |
| 分屏视图 | 支持 (>1440px) | 不支持 |
| Agent Inbox | 支持 | 不支持 |
| 配置导出/导入 | 支持 | 不支持 |
| Logto 登录 | 支持 | 不支持 |
