# Client SDK

`@clawlines/sdk` 是 Clawline 的客户端 SDK，让任何 JavaScript/TypeScript 应用都能与 OpenClaw Agent 建立 WebSocket 通信。

**核心能力：**
- 🔌 连接管理（自动重连、空闲超时）
- 💬 消息收发（文本、媒体、文件）
- 🤖 Agent 管理（列表、切换、上下文）
- ⌨️ 打字状态
- 📎 文件上传（HTTP multipart）
- 🔗 多连接池

**特点：**
- 纯 TypeScript，零运行时依赖
- 浏览器 + Node.js 双平台
- 类型安全的事件系统
